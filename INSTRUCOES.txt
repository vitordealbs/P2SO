==========================================
  SIMULADOR DE ESCALONAMENTO DE PROCESSOS
  Sistemas Operacionais - UFC
==========================================

COMO EXECUTAR:

1. FORMA MAIS SIMPLES (JAR já compilado):
   - Duplo clique em: executar.bat
   - Ou execute no terminal: java -jar target\P2SO-0.0.1-SNAPSHOT.jar

2. Para recompilar:
   - Duplo clique em: compilar.bat
   - Ou execute no terminal: mvn clean package -DskipTests

3. Acessar a interface:
   - Após executar, abra o navegador em: http://localhost:8080
   - A interface web será carregada automaticamente

==========================================
ARQUIVOS IMPORTANTES:

- config.txt: Configuração de quantum e aging
- README.md: Documentação completa do projeto
- target\P2SO-0.0.1-SNAPSHOT.jar: Arquivo executável

==========================================
CONFIGURAÇÃO:

Edite o arquivo config.txt para alterar:
  quantum:2    (tempo de quantum para Round-Robin)
  aging:1      (taxa de envelhecimento)

Ou configure diretamente na interface web.

==========================================
ENTRADA DE PROCESSOS:

Formato: tempo_chegada duração prioridade

Exemplo:
0 5 2
0 2 3
1 4 1
3 3 4

Cada linha representa um processo.

==========================================
ALGORITMOS IMPLEMENTADOS:

1. FCFS (First Come, First Served)
2. SJF (Shortest Job First)
3. SRTF (Shortest Remaining Time First)
4. Prioridade (sem preempção)
5. Prioridade (com preempção)
6. Round-Robin (quantum)
7. Round-Robin com Prioridade e Envelhecimento
